## Brief description of what each file do

NAME		: DESCRIPTION

acc5d.C		: Acceptance THnSparse, taking the leading pion (higher Zh), filling reco in two forms (using reco-variables and using generated (match) variables), WITHOUT good matching between reco and true histograms.
acc5d_NLP.C	: Acceptance THnSparse, NOT taking the leading pion (NLP), WITH good matching between reco and true histograms.
